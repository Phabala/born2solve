**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp
#include <iostream>


int   main()
{
      while(1) 
      {
            int d,e,b ;
            if (!(std::cin>>d))
                  break ;
            if (!(std::cin>>e))
                  break ;
            b = d + e;
            std::cout<<b<<std::endl;
      }
      return (0);
}
