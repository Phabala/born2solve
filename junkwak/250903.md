**문제 이름 또는 번호**: 별 찍기 - 2


**문제 링크**https://www.acmicpc.net/problem/2439

**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp
#include <iostream>

int   main()
{
      int a,b;
      std::cin>>a;
      for (int i = 1; i <= a; i++) {
            b = a;//1이면 별1개, 나머지 값을 다 띄어쓰기 , a값을 기준으로.
            for (int j = i; j < a; j++) {
                  std::cout<<" ";
            }
            for (int j = i; j > 0; j--){
                  std::cout<<"*";
            }
             std::cout<<"\n";
      }
      return (0);
}
