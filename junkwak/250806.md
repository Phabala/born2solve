**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp

#include <iostream>

using namespace std;
int   main()
{
      int i = 0;
      cin>>i;

      for (int j = 1; j <= 9; j++) {
            cout<<i<<" * "<<j<<" = "<<i * j<<endl;
      }
      return (0);
}
