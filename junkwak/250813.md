**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp

#include <iostream>

using namespace std;

int   main()
{
      int i , result;
      result = 0;
      i = 0;
      cin>> i;
      for(int j = 1; j <= i; j++)  {
            result += j;            
      }
      cout<<result<<endl;
      return (0);
}
