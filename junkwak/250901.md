**문제 이름 또는 번호**: 별 찍기 - 1


**문제 링크**https://www.acmicpc.net/problem/2438

**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp
#include <iostream>

int   main()
{
      int a,b;
      std::cin>>a;
      for (int i = 0; i <= a; i++) {
            b = i;
            for (int j = 0; j < b; j++)
                  std::cout<<"*";
            if (i != 0)
                  std::cout<<"\n";
      }
      return (0);
}
