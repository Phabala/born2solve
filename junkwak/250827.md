**오늘의 메모(선택)**: 

백준 레벨업을 위한 쉬운단계 푸는중입니다.


**코드(선택)**:

```cpp

#include <iostream>


int   main()
{
      int a,b;
      std::cin>>a;
      for(int i = 0; i < a; i++)
      {
            int d,e ;
            std::cin>>d;
            std::cin>>e;
            b = d + e;
            std::cout<<"Case #"<<i + 1<<": "<<b<<std::endl;
      }
      return (0);
}
