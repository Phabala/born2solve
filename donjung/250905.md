**오늘의 메모(선택)**: 
정수끼리 나눗셈을 하면 결과가 정수가 되고 소수점 이하는 버림되는 것을 놓침.
이를 해결하려면 나눗셈 연산에서 최소한 하나의 피연산자가 실수 타입이어야 한다.
캐스팅을 잘 생각하자.
**코드(선택)**:
``` c++
#include <iostream>
#include <vector>

using namespace std;

int	main()
{
	int	N;
	cin >> N;

	vector<int>	grades;
	int			max = 0;

	for (int i = 0; i < N; ++i) {
		int	score;
		cin >> score;
		grades.push_back(score);
		if (score > max)
			max = score;
	}

	float sum = 0;
	for (int i = 0; i < N; ++i) {
		float	new_score = 0;
		new_score = (float)grades[i] / max * 100;
		sum += new_score;
	}

	cout << sum / N << endl;

	return ( 0 );
}
```
