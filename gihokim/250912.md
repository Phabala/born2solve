**문제 플랫폼**: 백준

**문제 이름 또는 번호**: 2577 / 숫자의 개수

**문제 링크**: https://www.acmicpc.net/problem/2577

**오늘의 메모(선택)**: 하하...5 (어제 문제 처음에 Modular 연산인줄 몰라서 BigInteger 썼는데 BigInteger 안 써도 됐었다.)

**코드(선택)**:

```c++
#include <iostream>
#include <string>

using namespace std;

int ans[10];

int main() {
	int A, B, C;
	cin >> A >> B >> C;

	string s = to_string(A * B * C);
	for (int i = 0; i < s.length(); i++)
		ans[s[i] - '0']++;
	
	for (int i = 0; i < 10; i++)
		cout << ans[i] << '\n';
}
```
