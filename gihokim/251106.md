**문제 이름 또는 번호**: 1264 / 모음의 개수

**문제 링크**: https://www.acmicpc.net/problem/1264

**오늘의 메모(선택)**: 네 브론즈입니다.

**코드(선택)**:

```c++
#include <iostream>
#include <string>
#include <sstream>

using namespace std;

bool isVowel(char c) {
    return (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u' ||
            c == 'A' || c == 'E' || c == 'I' || c == 'O' || c == 'U');
}

int main() {
    while (true) {
        string s;
        getline(cin, s);
        if (s == "#") break;

        int ans = 0;
        for (int i = 0; i < s.length(); i++)
            if (isVowel(s[i])) ans++;
        cout << ans << '\n';
    }
}
```
