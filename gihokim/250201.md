**오늘의 메모(선택)**: 

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int main() {
    int N;
    cin >> N;
    
    if (N == 0) {
        cout << 1;
        return 0;
    }
    int ans = 1;
    while (N != 0) {
        ans *= N;
        N--;
    }
    cout << ans;
}
```
