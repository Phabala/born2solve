**문제 이름 또는 번호**: 1085 / 직사각형에서 탈출

**문제 링크**: https://www.acmicpc.net/problem/1085

**오늘의 메모(선택)**: bronze.

**코드(선택)**:

```c++
#include <iostream>
#include <algorithm>

using namespace std;

int main() {
    int x, y, w, h;
    
    cin >> x >> y >> w >> h;
    
    int ans = 21e8;
    ans = min(x - 0, ans);
    ans = min(w - x, ans);
    ans = min(y - 0, ans);
    ans = min(h - y, ans);

    cout << ans;
}
```
