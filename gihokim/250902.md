**문제 이름 또는 번호**: 11659 / 구간 합 구하기 4

**문제 링크**: https://www.acmicpc.net/problem/11659

**오늘의 메모(선택)**: 누적 합 문제

**코드(선택)**:

```c++
#include <iostream>
#include <string>

using namespace std;

int N, M;

int main() {
	// input & init
	cin >> N >> M;
	int memo[N + 1];

	memo[0] = 0;
	cin >> memo[1];
	for (int i = 2; i <= N; i++) {
		cin >> memo[i];
		memo[i] += memo[i - 1];
	}
	
	// solve
	string ans;
	int from, to;
	for (int i = 0; i < M; i++) {
		cin >> from >> to;
		ans += to_string(memo[to] - memo[from - 1]) += '\n';
	}

	// output
	cout << ans;
}
```
