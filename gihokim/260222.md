**오늘의 메모(선택)**: 문자열의 길이 이상의 횟수는 반복문을 반복 할 필요가 없다.

**코드(선택)**:

```c++
#include <iostream>
#include <vector>
#include <string>

using namespace std;

int main() {
    int N;
    cin >> N;

    vector<string> vec;
    
    string str = "SciComLove";
    
    while (vec.empty() || str != "SciComLove") {
        vec.push_back(str);
        str += str[0];
        str.erase(str.begin());
    }

    int i = N % vec.size();
    cout << vec[i];
}
```
