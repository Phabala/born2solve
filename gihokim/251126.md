**문제 링크**: https://www.acmicpc.net/problem/1267

**오늘의 메모(선택)**: 역시 쉬운 문제라도 그림 그리면서 풀어야 잘 된다.

**코드(선택)**:

```c++
#include <iostream>
#include <algorithm>

using namespace std;

int getCost(int len, int cost, int time) {
	return cost * (time / len + 1);
}

int main() {
	int N;
	cin >> N;
	
	int y = 0;
	int m = 0;

	for (int i = 0; i < N; i++) {
		int time;
		cin >> time;
		y += getCost(30, 10, time);
		m += getCost(60, 15, time);
	}

	if (y < m)
		cout << "Y ";
	else if (y > m)
		cout << "M ";
	else
		cout << "Y M ";

	cout << min(y, m);
}
```
