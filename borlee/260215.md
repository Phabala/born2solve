**오늘의 메모(선택)**: 
복습 겸 스택 기본 문제를 c++로 제출
짝지어서 성공하면 1, 아니면 0 반환. 이중반복문으로도 해결할 수 있지만, 스택은 O(N)이다. 
stack 메서드로 push, pop, top, empty, size가 존재한다.

**코드(선택)**
이 전에 제출했던 파이썬 코드다. 

```python
def solution(s):
    stack = []   
    for i in s:
        if stack and stack[-1] == i:
            stack.pop()          
        else:
            stack.append(i)
            
    return int(not stack)
```

```c++
#include <stack>
#include <string>

using namespace std;

int solution(string s)
{
    stack<char> st;
    
    for (int i = 0; i < s.length(); i++)
    {
        if (st.empty() || st.top() != s[i])
            st.push(s[i]);
        else
            st.pop();
    }
    return st.empty();
}
```
