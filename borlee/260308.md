**오늘의 메모(선택)**: 
파이썬으로 이전에 풀었던 것을 c++ 로 재체줄.
join 대신에 for문 += 연산을 사용했다.
이 문제를 풀면서 알았는데, 현재 보고 있는 책이 auto, range-based for loop(범위 기반 for문)을 꽤 많이 사용한다. 
그래서 코드가 더 깔끔한 경향도 있는데 
이 문법들은 c++11 부터 사용가능한 것이라고 한다.(OMG???).. 
C++98 제약 사항이 무척 많다.

**코드(선택)**
```c++
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

bool compare(const string& a, const string & b) {
    return a + b > b + a;
}

string solution(vector<int> numbers) {
    vector<string> strs;
    for (int num: numbers) {
        strs.push_back(to_string(num));
    }
    sort(strs.begin(), strs.end(), compare);
    
    string result = "";
    for (const string& s: strs) {
        result += s;
    }
    if (result[0] == '0') {
        return "0";
    }
    return result;
}
```
