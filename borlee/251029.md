문자열 자료형 파트에서 특정 문자열로 시작/끝나는 데 확인하는 함수가 있었다 (startswith/endswith) 

**코드(선택)**

```python
from typing import List

def solution(my_string: str, is_suffix: str) -> int:
    if my_string.endswith(is_suffix):
```
