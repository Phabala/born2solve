**오늘의 메모(선택)**: 
세로는 최소 3부터 시작
blue + yellow = S(넓이)
가로는 세로보다 같거나 크다.
-> return [가로(큰 값), 세로(같거나 작은 값)]
-> 세로는 넓이의 제곱근까지만 탐색해도 충분하다. (사각형 성립 조건)
근의 공식으로도 풀 수 있다고 한다. 와우 - 이건 다음주 

**코드(선택)**
```python
from typing import List, Optional, Tuple

def solution(brown: int, yellow: int) -> List[int]:

    total : int = brown + yellow
    for height in range(3, int(total **0.5) + 1): 
        if total % height == 0:
            width = total // height
            
            if (width - 2) * (height - 2) == yellow:
                return [width, height]
    return None
```
