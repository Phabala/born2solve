**오늘의 메모(선택)**: 
right는 counter, 왼쪽은 set.
왼쪽으로 옮기면서 양쪽 len이 같아지는 순간을 센다.
그리고 총 answer 반환
시뮬레이션 공부 중이라, 내용은 책 참고

**코드(선택)**
```python
from typing import List
from collections import Counter

def solution(topping: List[int]) -> int:
    answer: int = 0
    right_counter: Counter = Counter(topping)
    left_set: set = set()
    
    for t in topping:
        left_set.add(t)
        right_counter[t] -= 1
        if right_counter[t] == 0:
            del right_counter[t]
        if len(left_set) == len(right_counter):
            answer += 1
    return answer

```
