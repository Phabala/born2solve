**오늘의 메모(선택)**: 
bitset은 비트 시퀀스를 관리하는 컨테이너다. 책 코드는 컨테이너를 사용하는 코드였고 
이를 수동으로 구현했다. 2로 계속 나누고 역순으로 되돌리는 과정 추가.
return {transforms, removedZeros}로 끝내면 참 깔끔하고 좋을텐데..

for문에서 size_type 말고 size_t를 사용해도 되는데, 문자열 처리를 위한 것이라는 의도를 전달하기 좀 더 알맞다고 해서 사용했다.

**코드(선택)**
```c++
#include <string>
#include <vector>
// #include <bitset>
#include <algorithm>

using namespace std;

vector<int> solution(string s) {
    int transforms = 0;
    int removedZeros = 0;
    
    while (s != "1") {
        transforms++;
        int onesCount = 0;
        
        for (string::size_type i = 0; i < s.size(); ++i) {
            if (s[i] == '0') removedZeros++;
            else onesCount++;
        }
        
        s.clear();
        while(onesCount > 0) {
            s += (onesCount % 2 == 0 ? '0': '1');
            onesCount /= 2;
        }
        reverse(s.begin(), s.end());
    }
    vector<int> result;
    result.push_back(transforms);
    result.push_back(removedZeros);
    return result;
}
```
