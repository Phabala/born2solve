**오늘의 메모(선택)**: 
c++ 에서도 append가 python과 동일하게 존재하는 줄 알았다. 
그런데, c++에서는 컨테이너 string에 존재하며, 
문자열을 이어붙이는 데에만 사용 가능하다는 차이가 있어서 좀 신기했다. 
필수 문법 학습을 강의로 보고 있는데 가장 익숙한 코테 문제를 c++로
따라한 결과다. 


**코드(선택)**
```c++
#include <set>
#include <vector>

using namespace std;

vector<int> solution(vector<int> numbers) {
    set<int> sum;
    
    for (int i = 0; i < numbers.size(); ++i)
        for (int j = i + 1; j < numbers.size(); ++j)
            sum.insert(numbers[i] + numbers[j]);
    vector<int> answer(sum.begin(), sum.end());
    return answer;
}
```
