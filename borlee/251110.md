**오늘의 메모(선택)**: 
코딩 기초 트레이닝 

뜬금없이 타입에 대해 고민했다.
대입/할당 될 때 선언된 변수 타입이 우선된다. 
형변환도 꼭 하지 않아도 채점/제출이 되고 arr의 길이 ≤ 100'000 제한 이 있어서 문제될 것은 없긴 하다.


**코드(선택)**
```c
#include <stdio.h>
#include <stdbool.h>
#include <stdlib.h>

int* solution(int start_num, int end_num) {
    // return 값은 malloc 등 동적 할당을 사용해주세요. 할당 길이는 상황에 맞게 변경해주세요.
    int len;
    int *answer;
    
    len = start_num - end_num + 1;
    answer = malloc(sizeof(int) * len);
    if(!answer)
        return NULL;
    int i = 0;
    while (i < len)
    {
        answer[i] = start_num - i;
        i++;
    }
    return answer;
}
```

```c
#include <stdio.h>
#include <stdbool.h>
#include <stdlib.h>

// arr_len은 배열 arr의 길이입니다.
int solution(int arr[], size_t arr_len, int idx) {
    //만약 그러한 인덱스가 없다면 -1을 반환합니다.
    size_t i;
    if (idx < 0 || idx >= (int)arr_len)
        return -1;
    i = idx;
    while (i < arr_len)
    {
        if (arr[i] == 1)
            return (int)i;
        i++;
    }
    return -1;
}
```
