**오늘의 메모(선택)**: 
XOR(Exclusive or): 두 비트나 논리 값이 다를 때는 참, 같을 때는 거짓을 출력함
각 노드에서의 최소 공통 조상(LCA)를 찾으면 이게 문제에서 요구하는 만나는 라운드 수가 된다.
? 1을 빼는 이유는 , 사람에게 익숙한 1 시작이 아닌 기계가 다루기 쉬운 0 시작으로 변형하고 -> 2를 나누었을 때의 식이 간단해지기 떄문이다.

- XOR: 차이, 분기, 갈라짐
- >> : 단계 축소, 부모로 이동
- << : 크기 확장, 레벨 증가




**코드(선택)**
```c
int solution(int n, int a, int b)
{
    int x = (a - 1)^(b - 1);
    int round = 0;
    while (x > 0)
    {
        round++;
        x >>= 1;
    }
    return round;
}
```
