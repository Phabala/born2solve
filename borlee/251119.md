**오늘의 메모(선택)**: 
탐색이 끝나면 되돌리기. visited[i] = True, False 부분이 핵심이다

**코드(선택)**
```python
from typing import List, Tuple

def explore(current_energy: int, cleared_count: int, dungeons: List[Tuple[int, int]], visited: List[bool]) -> int:
    max_result : int = cleared_count
    for i in range(len(dungeons)):
        required, cost = dungeons[i]
        
        if current_energy >= required and not visited[i]:
            visited[i] = True
            result: int = explore(current_energy - cost, cleared_count + 1, dungeons, visited)
            max_result = max(max_result, result)
            visited[i] = False 
    return max_result

def solution(k: int, dungeons: List[Tuple[int, int]]) -> int:
    visited: List[bool] = [False] * len(dungeons)
    return explore(k, 0, dungeons, visited)
```
