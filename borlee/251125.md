**오늘의 메모(선택)**: 
dfs(idx, total) 각 숫자에 +, - 를 붙여 완전탐색한다
재귀 호출이 반환하는 두 선택 결과(+/-)를 더해서, 전체 경우의 수를 계산한다.

**코드(선택)**
```python
from typing import List

def solution(numbers: List[int], target: int) -> int:
    n: int = len(numbers)

    def dfs(idx: int, total: int) -> int:
        if idx == n:
            return 1 if total == target else 0 # 기저 조건
        count_plus = dfs(idx + 1, total + numbers[idx])
        count_minus = dfs(idx + 1, total - numbers[idx])
        return count_plus + count_minus
    return dfs(0, 0)
```
