**문제 이름 또는 번호**: 단어순서 뒤집기

**문제 링크**: https://www.acmicpc.net/problem/12605

**오늘의 메모(선택)**:

**코드(선택)**:
```cpp
#include <iostream>
#include <stack>
#include <sstream>

int	main()
{
	int	N;
	std::string	line;
	std::string	word;

	std::cin >> N;
	std::cin.ignore();
	for (int i = 0; i < N; ++i)
	{
		std::getline(std::cin, line);
		std::stringstream	ss(line);
		
		std::stack<std::string>	stack;
		while (ss >> word)
			stack.push(word);

		std::cout << "Case #" << (i + 1) << ":";
		while (stack.empty() == false)
		{
			std::cout << " " << stack.top();
			stack.pop();
		}
		std::cout << "\n";
	}

	return 0;
}
```
