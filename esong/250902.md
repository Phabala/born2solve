**문제 이름 또는 번호**: 좋은 단어

**문제 링크**: https://www.acmicpc.net/problem/3986

**오늘의 메모(선택)**:

**코드(선택)**:
```cpp
#include <iostream>
#include <stack>

int	main()
{
	int	inputCnt;
	std::stack<char>	stack;
	std::string	input;

	std::cin >> inputCnt;
	int	result = 0;
	while (inputCnt--)
	{
		std::cin >> input;
		std::stack<char>	stack;
		for (int i = 0; input[i]; ++i)
		{
			if (stack.size() > 0 && stack.top() == input[i])
				stack.pop();
			else
				stack.push(input[i]);
		}
		if (stack.size() == 0)
			++result;
	}
	std::cout << result;

	return 0;
}
```
