**오늘의 메모(선택)**: 

bitset으로 N의 1 비트 개수를 세면 최소 점프 횟수를 바로 구할 수 있음

**코드(선택)**: 

```c++
#include <bitset>

using namespace std;

int solution(int N) {
  return bitset<32>(N).count();
}
```
