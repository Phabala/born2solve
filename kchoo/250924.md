**오늘의 메모(선택)**: 

그리디 문제

**코드(선택)**: 

```c++
#include <vector>
#include <algorithm>
using namespace std;

int solution(vector<int> d, int budget) {
  sort(d.begin(), d.end());
  int count = 0;

  for (int amount : d) {
    if (budget < amount) break;
    budget -= amount;
    count += 1;
  }
  return count;
}
```
