**문제 이름 또는 번호**: 2xn 타일링

**문제 링크**: https://school.programmers.co.kr/learn/courses/30/lessons/12900

**오늘의 메모(선택)**: 

**코드(선택)**: 

```c++
#include <vector>
using namespace std;

long long solution(int n) {
  if (n == 1) return 1;
  if (n == 2) return 2;

  vector<long long> dp(n + 1, 0);
  dp[1] = 1;
  dp[2] = 2;
  for (int i = 3; i <= n; i++) {
    dp[i] = (dp[i - 1] + dp[i - 2]) % 1000000007;
  }
  return dp[n];
}
```
