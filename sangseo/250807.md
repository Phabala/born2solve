**오늘의 메모(선택)**: 문자열 리스트 or 이터레이터를 인자로 받아 원소들을 연결하여 하나의 문자열로 만드는 join() 함수를 사용해보았다.
''.join(리스트) 형식으로 사용하는데, '' 안에 연결에 쓰일 구분자를 넣으면 리스트들을 구분자로 연결하여 하나의 문자열로 만드는 식이다.

**코드(선택)**:
```python
n = list(input())
cnt = len(n)
total = 0
for i in range(cnt):
    n.insert(0, n[-1])
    n.pop()
    total += int(''.join(n))
print(total)
```
