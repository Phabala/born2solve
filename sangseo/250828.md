**오늘의 메모(선택)**: 결국 못품.. 내일 마저하자..

**코드(선택)**:
```python
n, k = map(int, input().split())
array = []

for i in range(n):
    array.append(list(map(int, input().split())))
for i in range(n-1, -1, -1):
    for j in range(n-1, -1, -1):
        for kk in range(k-1):
            array[i].insert(j, array[i][j])
    for kk in range(k-1):
        array.insert(i, array[i])
for i in range(n*k):
    for j in range(n*k):
        print(array[i][j], end=' ')
    print()
```
