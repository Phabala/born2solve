**오늘의 메모(선택)**: 집합 문제였다. 파이선의 set 자료형과 set의 메소드를 활용해 쉽게 풀었다.

**코드(선택)**:
```python
n, m = map(int, input().split())
a = set()
for i in range(n):
    a.add(input())
b = set()
for i in range(m):
    b.add(input())
tmp = list(a.intersection(b))
tmp.sort()
print(len(tmp))
for name in tmp:
    print(name)
```
