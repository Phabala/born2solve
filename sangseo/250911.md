**오늘의 메모(선택)**: 전에 못푼 문제 해결!

**코드(선택)**:
```python
import sys
from collections import deque

input = sys.stdin.readline
t = int(input())
for i in range(t):
    n, p = map(int, input().split())
    deq = deque(map(int, input().split()))
    cnt = 0
    idx = p
    while (deq):
        maxdeq = max(deq)
        cur = deq.popleft()
        if cur < maxdeq:
            deq.append(cur)
            if idx == 0:
                idx = len(deq)
            idx -= 1
        else:
            cnt += 1
            if idx == 0:
                print(cnt)
                break
            idx -= 1
```
