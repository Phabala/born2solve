**오늘의 메모(선택)**: stack을 활용하는 문제였다. 잘하면 변수 몇개와 스택 하나로만 풀 수 있었을것 같은데 종이 없이 머리로만 하려니 복잡해서 때려치우고 리스트 두개로 풀었다. 스택하나는 다음에 도전해봐야지..

**코드(선택)**:
```python
n = int(input())
l = []
s = []
flg = 1
for i in range(n):
    l.append(int(input()))
stack = []
for i in range(1,n+1):
    stack.append(i)
    s.append('+\n')
    while (stack and l and stack[-1] == l[0]):
        stack.pop()
        l.pop(0)
        s.append('-\n')
    if (l and stack and l[0] < stack[-1]):
        flg = 0
        print('NO')
        break
if flg:
    print(''.join(s))
```
