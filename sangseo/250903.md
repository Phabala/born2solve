**문제 이름 또는 번호**: 18110 / solved.ac

**오늘의 메모(선택)**: 진짠지 아닌지 몰겠지만 solved.ac의 백준 문제 난이도 계산 방식에 대한 문제였다. 그냥 간단한 수학, 구현, 정렬 문제인듯.

**코드(선택)**:
```python
import sys
input = sys.stdin.readline
n = int(input())
if n == 0:
    print(0)
    exit()
avrg = n / 100 * 15
if (avrg // 0.5) % 2:
    avrg = int(avrg) + 1
else:
    avrg = int(avrg)
l = []
for i in range(n):
    l.append(int(input()))
l.sort()
l = l[avrg:n-avrg]
ret = sum(l) / (n-avrg*2)
if (ret // 0.5) % 2:
    ret = int(ret) + 1
else:
    ret = int(ret)
print(ret)
```
