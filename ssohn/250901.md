**오늘의 메모(선택)**: 레전드 그냥 분수 만들어서 최대 공약수로 약분하면 될 것을 일일이 하나씩 분수를 만들어서 다시 합치고 또 약분하려고 난리를 쳤다. 역시 사람은 수학공부를 해야..

참고 :

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int gcd (int A, int B) {
	if (B == 0) {
		return (A);
	}
	return (gcd(B, A % B));
}

int main() {
	int A, B, C, D;
	cin >> A >> B;
	cin >> C >> D;

	int numerator = A * D + C * B;
	int denominator = B * D;

	int g = gcd(numerator, denominator);
	cout << numerator / g << ' ' << denominator / g;
	return (0);
}
```
