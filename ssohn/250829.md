**문제 링크**: https://www.acmicpc.net/problem/13241

**오늘의 메모(선택)**: 어제 푼 최소 공배수와 비슷한 문제인데 범위가 long long인..

참고 :

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

long long gcd(long long a, long long b) {
	if (b == 0) {
		return (a);
	}
	return (gcd(b, a % b));
}

long long lcm(long long a, long long b) {
	return ((a / gcd(a, b)) * b);
}

int main() {
	long long A, B;
	cin >> A >> B;
	cout << lcm(A, B) << '\n';
}
```
