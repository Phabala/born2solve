**오늘의 메모(선택)**: 오늘은 계수 정렬을 공부했다. 중간에 좀 메모리 관련해서 많은 일이 있어서 많이 날려먹었는데 계수 정렬 신기하다.

참고 :

**코드(선택)**:

```c++
#include <iostream>
#include <vector>

using namespace std;

int arr[10001];
int main() {
	int N;
	int num;

	cin >> N;

	for (int i = 0; i < N; i++) {
		cin >> num;
		arr[num]++;
	}
	for (int i = 1; i <= 10000; i++) {
		while (arr[i] > 0) {
			cout << i << '\n';
			arr[i]--;
		}
	}
	return (0);
}

```
