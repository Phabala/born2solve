**오늘의 메모(선택)**: 파이썬 최고

**코드(선택)**:

```python
def solution(target):
    n = [0]
    
    for start, end in sorted(target):       
        if n[-1] <= start: n.append(end)
        else: n[-1] = min(n[-1], end)
        
    return len(n)-1
```
