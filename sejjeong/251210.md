                큰 수 만들기

**문제 링크**: https://school.programmers.co.kr/learn/courses/30/lessons/42883


**오늘의 메모(선택)**: 
            erase 없이 만드는 방법을 생각 중인데... 괜찮은 방법이 없을까

**코드(선택)**:



```C++

#include <string>
#include <vector>
#include <cstdlib>
#include <iostream>
#include <unordered_set>
#include <queue>

using namespace std;

string solution(string number, int k) {
    // 앞을 자르는 것과 중간에 자르는 것 중에 무엇이 더 좋은 지
    // 100만 자리라면 조합 갖고는 안되지
    // 비교 연산해도 100만 번 사용해야 해서 아웃

    // 앞자리를 제일 크게 하려면 어떻게 해야 하는가
    // 두번째 자리를 제일 크게 하려면 어떻게 해야 하는가
    // 이렇게 
    
    // 앞자리가 큰 게 최적의 해임
    // 한 번에서 두 번만 훑어 보면 되긴 할 거 같다.

    int i = 0;
    while (k > 0 && i < number.size() - 1)
    {
        if (number[i] < number[i + 1])
        {
            // erase 하고 싶지 않은데 
            number.erase(number.begin() + i);
            --k;
            i = 0;
            continue;
        }
        ++i;
    }
    
    number = number.substr(0, number.size() - k);
    
    return number;
}

```
