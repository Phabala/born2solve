**오늘의 메모(선택)**: 
                정렬 후, 투포인터로 계산 N^2에서 N log N으로 줄음

**코드(선택)**:

C++

```

#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<int> people, int limit) {
    int count = 0;
    
    sort(people.begin(), people.end());
    
    int left = 0;
    int right = static_cast<int>(people.size() - 1);
    while (left < right)
    {
        if (people[right] + people[left] <= limit)
        {
            --right;
            ++left;
            ++count;
        }
        else
        {
            --right;
            ++count;
        }
    }
    
    if (left == right)
    {
        ++count;
    }
    
    return count;
}
```
