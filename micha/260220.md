**오늘의 메모(선택)**:

입력: [-16, -8, -4]
출력:-32 (-2가 정답)
등비가 소수일때 문제 발생

입력값 〉	[-16, -8, -4]
기댓값 〉	-2
실행 결과 〉	실행한 결괏값 0이 기댓값 -2과 다릅니다.

common[i]와 common[i-1]는 둘 다 int라서, Java는 먼저 int / int 를 int로 계산함

**코드(선택)**:

```java

//오답
class Solution {
    public int solution(int[] common) {
        int answer = 0;
        int num1 = 0;
        int num2 = 0;
        int check = 0; // 0 등차, 1 등비
        int common_length = 0;

        common_length = common.length;

        int i = 0;
        num1 = common[i+1] - common[i];
        i++;
        num2 = common[i+1] - common[i];

        if (num1 != num2)
            check = 1;

        if (check == 0)
            answer = common[common_length-1] + num1;
        else 
        {
            num1 = common[i] / common[i - 1]; // 
            answer = common[common_length - 1] * num1;
        }

        return answer;
    }
}
```
