**오늘의 메모(선택)**: 
내 방식 (오류) : 매칭이 실패 했을때 한 글자씩 다시 찾는 방식 -> 부분 문자열로 어디인가 있을때 사용하는 방식임.
수정 방식 (해결) : startsWith() 메서드를 사용해 단어가 매칭되는지 확인하고, 매칭이 되면 인덱스를 단어 길이만큼 증가시키는 방식으로 변경

a.startWith(b,c) : 문자열 a의 c번째 인덱스 부터 b문자열이 매칭되는지 확인하는 메서드

s.startsWith("ma", 2)
== s.substring(2, 2 + "ma".length()).equals("ma")

**코드(선택)**:

```java
class Solution {
    
    public int solution(String[] babbling) {
        int answer = 0;
        
        for (int i = 0; i < babbling.length; i++)
        {
            if (check_word(babbling[i]) == 1)
                answer++;
        }
        
        return answer;
    }
    
    public int check_word(String babbling)
    {
        String[] word = {"aya", "ye", "woo", "ma"};
        int i = 0;
        
        while (i < babbling.length())
        {
            boolean match = false;

            for (int j = 0; j < word.length; j++)
            {
                if (babbling.startsWith(word[j], i))
                {
                    i += word[j].length();
                    match = true;
                    break;
                }
            }
            if (!match)
                return 0;

        }
        return 1;
    }
}


//오류 방식
class Solution {
    
    public int solution(String[] babbling) {
        int answer = 0;
        
        for (int i = 0; i < babbling.length; i++)
        {
            if (check_word(babbling[i]) == 1)
                answer++;
        }
        
        return answer;
    }
    
    public int check_word(String babbling)
    {
        String[] word = {"aya", "ye", "woo", "ma"};
        int j = 0;
        while (true)
        {
            for (int i = 0; i < word.length; i++)
            {
                if (babbling.substring(j, (j + word[i].length())).equals(word[i]))
                {
                    j += word[i].length();
                    if (j >= babbling.length())
                    {
                        break;
                    }
                }
            }
            if (j == babbling.length())
                return 1;
            j++;
            if (j >= babbling.length())
            {
                break;
            }
        }
        return 0;
    }
}
