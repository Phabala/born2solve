**오늘의 메모(선택)**: 와 다른사람 풀이에서 [5 - num%2]이부분 보고 머리를 탁 쳤습니다. 세상은 넓고 고수는 많다..

**코드(선택)**:

```c++
#include <stdio.h>
#include <stdbool.h>
#include <stdlib.h>

char* solution(int num) {
    // 리턴할 값은 메모리를 동적 할당해주세요
    
    // int len = 4;
    // if (num % 2 == 0)
    //     len = 5;
    char* answer = (char *)malloc(5 - num%2);
    answer = "Odd";
    if (num % 2 == 0)
        answer = "Even";
    return answer;
}
```
